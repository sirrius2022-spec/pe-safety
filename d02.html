불안전 행동 vs 불안전 상태
88% vs 10% — 시험 단골
▼
불안전 행동 (88%) — 인적 원인
  • 안전장치 무효화·제거
  • 보호구 미착용
  • 위험장소 접근·출입
  • 불안전한 속도·자세로 작업
  • 지시·절차 미준수
불안전 상태 (10%) — 물적 원인
  • 방호장치 결함·미설치
  • 작업환경 불량 (조명·환기·소음)
  • 복장·보호구 결함
  • 작업장소 불량 (정리정돈 미흡)
  • 기계·설비 노후화
🔢 숫자 암기
88% 행동 + 10% 상태 + 2% 천재지변 = 100%
→ "팔팔한 사람(행동)이 문제의 대부분!"

활성 실패 vs 잠재 실패
시험 단골 개념
▼
활성 실패 (Active Failure)
  • 현장 작업자의 즉각적 행동으로 인한 실패
  • 즉시 결과가 나타남
  • 예) 안전장치 해제, 보호구 미착용
잠재 실패 (Latent Failure)
  • 조직·관리 수준의 숨겨진 결함
  • 시간이 지나 활성화됨
  • 예) 부실한 안전교육, 과도한 업무량
💡 핵심
잠재 실패가 더 위험!
→ 겉으로 안 보이다가 어느 순간 터짐
→ 중대재해처벌법의 경영책임자 의무와 연결

재해 원인 분류
직접원인 vs 간접원인
▼
직접원인 (불안전 행동·상태)
  • 불안전 행동 — 인적 원인 (88%)
  • 불안전 상태 — 물적 원인 (10%)
간접원인 (4M)
  • Man — 인적 요인 (지식·기능·태도 부족)
  • Machine — 기계·설비 결함
  • Media — 작업환경·방법 불량
  • Management — 관리적 요인
💡 4M 암기팁
"사람·기계·환경·관리" = 4M
→ Man / Machine / Media / Management
→ 위험성평가 원인 분석 시 4M으로 분류!

▼

  • 치료비·보상금

간접비 (수면 아래) — 4
  • 생산 손실, 시간 손실
  • 기계·설비 손상
  • 작업 중단, 공기 지연
  • 신규 인력 채용·훈련비
  • 사기 저하, 이미지 손실

→ "보이는 비용보다 숨어있는 비용이 4배 크다"
→ 총재해코스트 = 직접비 × 5

조직·관리 수준에 숨겨진 결함
