  • ① Engineering (기술) — 안전설계·방호장치·기계화

  • ② Education (교육) — 안전보건교육·훈련

4M (재해 간접원인 분류)
  • Man — 인적 요인 (지식·기능·태도)
  • Machine — 기계·설비 결함
  • Media — 작업환경·방법 불량

Human Error — 인간의 오류
착오요인 3단계 + 분류
▼
착오요인 3단계 (정보처리)
  • ① 인지 단계 — 정보 받아들임 오류 (잘못 봄·들음)
  • ② 판단 단계 — 의사결정 오류 (잘못된 판단)
  • ③ 조작 단계 — 행동 실행 오류 (실수·망설임)
Swain·Gattman 분류 (심리적)
  • Omission Error — 누락 (해야 할 일 안 함)
  • Commission Error — 작위 (잘못된 행위)
  • Sequential Error — 순서 오류
  • Time Error — 시간 오류 (너무 빠름·늦음)
  • Extraneous Error — 불필요한 행위
James Reason 분류 (원인적)
  • Slip — 의도와 다른 행동 (실수)
  • Lapse — 망각·기억 실수
  • Mistake — 잘못된 의사결정
  • Violation — 의도적 위반
💡 시험 포인트
Reason의 분류 — 스위스치즈 모델과 연결
→ Slip·Lapse = 활성실패 / Violation = 잠재실패

Fail Safe — 페일 세이프
★ 시험 핵심 — 고장이 나도 안전
▼
⚡ 핵심 개념
기계·시스템이 고장 나도 안전한 상태로 자동 전환
→ "고장이 일어나도 안전이 유지된다"
Fail Safe 3단계 (기능적 분류)
  • ① Fail Passive — 고장 시 정지 (가장 기본)
    예) 신호기 고장 시 빨간불 점등
  • ② Fail Active — 고장 시 경보 후 단시간 운전
    예) 가스 누출 시 경보 후 차단
  • ③ Fail Operational — 고장 시 다음 점검까지 안전 운전
    예) 비행기 다중 시스템
건설현장 적용 사례
  • 승강기 — 와이어 절단 시 자동 정지
  • 크레인 — 과부하 시 자동 정지
  • 가스 검지기 — 누출 시 자동 차단
  • 전기 — 누전차단기 자동 작동
💡 연상
"Fail = 고장, Safe = 안전"
→ 고장(Fail) 나도 안전(Safe) 보장!
🚫
Fool Proof — 풀 프루프
★ 시험 핵심 — 잘못 사용해도 안전
▼
⚡ 핵심 개념
작업자가 실수해도 사고가 일어나지 않도록 설계
→ "바보(Fool)도 못 망가뜨리는(Proof) 설계"
Fool Proof 5대 기능
  • ① 격리 (Isolation) — 위험원과 분리
    예) 회전체 덮개
  • ② 인터록 (Interlock) — 조건 미충족 시 동작 불가
    예) 출입문 닫혀야 작동
  • ③ 자물쇠 (Lockout) — 위험 조작 잠금
    예) 정비 중 전원 잠금
  • ④ 트립 (Trip) — 위험 시 자동 차단
    예) 누전차단기
  • ⑤ 오버런 (Overrun) — 정지 후 잔류 동작 방지
    예) 회전체 완전 정지 후 개방
