감전의 위험성
  • 전류 1mA — 찌릿한 느낌
  • 전류 10mA — 근육 수축·이탈 불가

1찌릿 → 10못빠짐 → 50사망위험 → 100확실사망
→ 안전전압 50V = 생사의 기준

화재 종류별 소화방법
ABC 분류 — 리튬화재 최신 추가!
▼
  • A급 (일반화재) — 목재·종이·섬유
    → 물·ABC분말소화기
  • B급 (유류화재) — 기름·가연성 액체
    → CO₂·포·분말소화기 (물 금지!)
  • C급 (전기화재) — 전기기계·배전반
    → CO₂·분말 (물 절대 금지!)
  • D급 (금속화재) — 마그네슘·나트륨
    → 건조사(모래) (물 금지!)
  • K급 (주방화재) — 식용유
    → 강화액 소화기
⚡ 리튬배터리 화재 특별 주의 (2024 화성)
리튬화재 = 물 사용 절대 금지!
→ 물 접촉 시 폭발적 반응
→ 다량의 물로 냉각 또는 질식소화
→ 리튬화재용 특수소화기 비치 필요
→ 스프링클러 미작동 → 화성 사고 원인

밀폐공간이란?
  • 환기 불충분한 밀폐된 장소
  • 산소 결핍·유해가스 발생 우려 장소

"구하러 들어간 사람도 쓰러진다"
→ 동반사고 비율 매우 높음
→ 2차 피해 방지 = 공기호흡기 착용 후 진입
→ 절대 맨몸으로 구조 진입 금지!
